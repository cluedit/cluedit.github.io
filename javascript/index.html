        # Javascript 시작하기

        # Javascript Fundamentals

        JavaScript는 동적 프로그래밍 언어, 객체 기반의 스크립트 프로그래밍 언어입니다. 주로 웹 브라우저 또는 node.js와 같은 런타임 환경에서 사용하고 있습니다. ECMAScript (혹은 ES)에 대한 ECMA-262을 표준으로 합니다.

        # ECMAScript

        ECMAScript (혹은 ES)는 javascript의 기반이 되는 script programming language이며, Ecma International 표준화 기구에 의해서 ECMA-262 및 ECMA-402 스펙에 표준화되었습니다. ECMAScript (혹은 ES)는 쉽게 말해서 자바스크립트의 표준화된 버전입니다! javascript는 이 ECMAScript와의 호환을 목표로 하면서, 발전하고 있습니다.

        # Webpack

        node.js의 등장으로 실질적인 모듈화가 가능해지면서, npm, yarn과 같은 package manager들과 함께 Javascript도 새로운 시대를 맞이하게 되었습니다.

        하지만, node.js 환경이 아닌 웹 브라우저(Web Browser) 환경에서 파일 단위의 모듈(module)을 사용하는 일은 여전히 쉬운 일이 아닙니다. node.js 환경에서 제작한 Web application을 웹 브라우저(Web Browser) 환경의 사용자들에게 배포하기 위해 의존성을 갖는 모듈들을 즉시 실행 함수 표현(IIFE)로 변경해 주면서 하나의 파일로 묶어주는 과정(bundling)은 필수적이고, 이 과정을 쉽게 처리해주는 도구가 바로 Webpack입니다.

        # Eslint

        Lint 혹은 Linter는 요즘 프로그래밍을 하다보면, 아주 쉽게 접할 수 있는 용어입니다. 오류를 찾는 것도 포함하지만, 일반적으로는 특정 Programming Style에 맞지 않는 source code를 찾는 도구를 말합니다. eslint는 javascript에서 사용할 수 있는 Linter입니다.

        # Babel

        Babel 또는 Babel.js는 웹 어플리케이션 개발에 사용되는 오픈 소스 도구입니다. Babel을 이용하면, 프로그래머가 원하는 현대 언어 코드를 작성한 뒤, 일반적인 javascript 환경에서 실행할 수 있는 javascript로 변환할 수 있습니다. 덕분에, 환경에 구애받지 않고 정상적으로 작동하는 javascript 프로그램을 작성할 수 있습니다.
